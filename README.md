# AI Dev Platform

AI Dev Platform is a modular, open-source toolkit to help developers
analyze, verify, and improve AI-generated code.

## Features

- Syntax analysis
- Style checks
- Security heuristics
- CLI tool & Web UI
- GitHub Action integration
- Modular architecture
